# Google Pay on the Web: RESTful API Integration

Merchants need to follow the below steps in order to integrate Google Pay with their website.

## Step 1: Configure Google Pay on the Web

The merchant will need to be configured with the merchant environment to accept [Google Pay on the web](https://developers.google.com/pay/api/web/guides/tutorial). This includes define Google Pay API Version, request a payment token, define payment card networks & auth methods, describe payment allowed methods, add payment tag, determine readniness to pay, dd google pay button etc.

- When using *GooglePay*, the merchant will utilize the certificate generated by Commerce Hub. Please contact our Account Representative in order to get the certificate.
- When using *WalletDecrypted*, the merchant will utilize their own certificate.
